🌳 Projeto Árvores e Algoritmos

Este projeto é uma ferramenta visual para entender como funcionam algumas estruturas de dados (Árvores) e como resolver problemas de logística (como o do Caixeiro Viajante) usando programação.

📝 O que este projeto faz?

Desenha Árvores: Você insere números e vê a árvore sendo montada na tela em tempo real.

Balanceamento Automático: Mostra como a tecnologia organiza os dados para que a busca seja sempre rápida (modelos AVL e Rubro-Negra).

Rota de Cidades (TSP): O sistema cria várias cidades no mapa e calcula o caminho mais curto para passar por todas elas.

Testes de Velocidade: Uma aba que mede quantos milissegundos o computador leva para processar grandes quantidades de dados.

🛠️ Linguagem utilizada

HTML, CSS e JavaScript: Não precisa instalar nada para rodar o projeto, ele funciona direto no seu navegador (Chrome, Edge ou Firefox).
